 # Gradle Task
 - makedist : 배포전에 dist 디렉터리에 최종 빌드
 - dist : makedist를 실행하고 dist 디렉터리를 서버로 배포. dist 후에는 서버에서 애플리케이션이 reload 된다.
 - distweb : 웹 리소스만 배포할때 사용
 - clear  : build 와 dist 디렉터리 삭제
 - gradle dist / gradle clear dist / gradle distweb
